# vue-admin-template-i18

基于官方的：https://github.com/PanJiaChen/vue-admin-template


已修改为中文版，方便自己后续进行扩展开发，仅供学习。


## 相关项目

- [vue-element-admin](https://github.com/PanJiaChen/vue-element-admin)

- [electron-vue-admin](https://github.com/PanJiaChen/electron-vue-admin)

## Build Setup

```bash
# 克隆项目
git clone https://github.com/894658027/vue-admin-tempalte-i18.git

# 进入项目目录
cd vue-admin-tempalte-i18

# 安装依赖
npm install

# 建议不要直接使用 cnpm 安装以来，会有各种诡异的 bug。可以通过如下操作解决 npm 下载速度慢的问题
npm install --registry=https://registry.npm.taobao.org

# 启动服务
npm run dev
```

浏览器访问 [http://localhost:9528](http://localhost:9528)

## 发布

```bash
# 构建测试环境
npm run build:stage

# 构建生产环境
npm run build:prod
```
